# Foundry models benchmark

이 디렉터리에는 모델 코딩 벤치마크 작업의 코드, 원본 자료, 최종 산출물이 들어 있습니다.

## 구성

- `reports/` - 최종 정리 리포트 
- `benchmarks/` - 공개 벤치마크 기준표
- `research/` - 벤치마크 리서치에 사용한 원본/참고 자료.
- `coding-bench/` - 벤치마크 하네스, task, 보존된 실행 산출물. `custom/`(커스텀 코딩 평가)과 `swe-bench/`(SWE-bench Lite/Verified)로 나뉩니다.

## 보존된 실행 산출물

- `coding-bench/custom/runs/first-pass/` - 1차 커스텀 코딩 평가.
- `coding-bench/custom/runs/repeat-5x-all-models/` - 전체 모델 5회 반복 커스텀 코딩 평가 요약.
- `coding-bench/swe-bench/runs/clean-swe-10-v2/` - 최종 정리된 SWE-bench Lite 10개 샘플 실행 결과.
